* 关于import导入
1. 直接用import导入包
    ``` Go
      import "fmt"
   ```
    在多个包的情况下
    ``` Go
       import (
          "fmt"
          "math"
        )
    ```
2. 用别名导入（或直接用点）
    ``` Go
       import a "fmt"//之后需要输入fmt的地方可以用a代替
    
    ```
